# Mamba
A minimal Ubuntu-based container image that ships with Miniforge (conda) and `mamba`, intended as a clean base for Python/ML workloads and reproducible builds. This [image](https://hub.docker.com/repository/docker/hieupth/mamba) follows a **monthly release** scheme aligned with NVIDIA container tags.
