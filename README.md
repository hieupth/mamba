# Mamba
Simple image of miniforge3 (conda|mamba) with CUDA and CuDNN supported.
## Available image tags
There are two image tags:
* **hieupth/mamba** - cpu only
* **hieupth/mamba:cuda** - latest CUDA (12.8) and CuDNN (9.8).
